La formule de Kellemerman-Klein s'écrit

Ft = C / (0.161 * P) + (μt * D2/1.715) + (μh *Dh/2)


Avec :

C = Couple de Serrage (souvent en N.m) ici en N.mm

P = Pas de vis 

μt = Coefficient de frottement filet 

μh = Coefficient de frottement tete

Dh = (Diamètre sous tête + Diamètre de passage ) / 2

D2 = diam_nominale - (0.6495 * Pas de vis)
	​


Trou de passage :
C'est le trou percé dans la pièce non taraudée, qui permet à la vis de passer librement — sans que ses filets accrochent, toutes les vis n'ont pas de trou de passage, certaines sont entièrement fileté

Tête de vis
   ↓
┌──────────────┐  ← Pièce supérieure percée (trou de passage)
│              │
│   Ø passage  │   ← un peu plus grand que le diamètre nominal
│              │
├──────────────┤  ← Pièce inférieure taraudée (ou écrou)
│   Filetage   │
└──────────────┘

La norme ISO 68-1 définit les principes de conception des filetages métriques. Les filets métriques ISO sont des filets symétriques en forme de V. La forme du filet en V a un angle de flanc de 60°, et les filets mâle
